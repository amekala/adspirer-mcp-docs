# LinkedIn Ads Tools

| Tool | Description |
|------|-------------|
| `select_linkedin_campaign_type` | Choose campaign type (Image, Text, Video, Carousel) |
| `create_linkedin_image_campaign` | Create a sponsored image campaign |
| `create_linkedin_text_campaign` | Create a text ad campaign |
| `create_linkedin_video_campaign` | Create a video ad campaign |
| `create_linkedin_carousel_campaign` | Create a carousel ad campaign |

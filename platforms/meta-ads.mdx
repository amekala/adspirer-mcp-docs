# Meta Ads Tools

Adspirer provides **20 tools** for Meta Ads management across Facebook and Instagram.

## Campaign Creation

| Tool | Description |
|------|-------------|
| `select_meta_campaign_type` | Choose campaign type (Image, Carousel, Video) |
| `create_meta_image_campaign` | Create a single-image ad campaign |
| `create_meta_carousel_campaign` | Create a carousel ad campaign |
| `create_meta_video_campaign` | Create a video ad campaign |
| `duplicate_meta_campaign` | Duplicate an existing campaign |

## Campaign Management

| Tool | Description |
|------|-------------|
| `list_meta_campaigns` | List all Meta campaigns |
| `get_meta_campaign_details` | Get detailed campaign configuration |
| `update_meta_campaign` | Update campaign settings |
| `pause_meta_campaign` | Pause a campaign |
| `resume_meta_campaign` | Resume a paused campaign |

## Ad Sets & Ads

| Tool | Description |
|------|-------------|
| `list_meta_ad_sets` | List ad sets within a campaign |
| `add_meta_ad_set` | Add a new ad set |
| `update_meta_ad_set` | Update ad set targeting or budget |
| `list_meta_ads` | List ads within an ad set |
| `add_meta_ad` | Add a new ad creative |
| `update_meta_ad` | Update ad content |

## Performance & Insights

| Tool | Description |
|------|-------------|
| `get_meta_campaign_performance` | Get campaign metrics and ROAS |
| `get_meta_ad_creatives` | View creative assets and their performance |
| `analyze_meta_ad_performance` | AI analysis of ad creative effectiveness |
| `analyze_meta_wasted_spend` | Identify budget waste and optimization opportunities |
| `explain_meta_anomaly` | AI-powered explanation of performance changes |
| `detect_meta_creative_fatigue` | Detect when ad creatives need refreshing |
| `get_meta_audience_insights` | Audience demographic and interest insights |
| `analyze_meta_audiences` | Deep audience analysis and recommendations |
| `optimize_meta_budget` | Budget optimization recommendations |
| `optimize_meta_placements` | Placement optimization (Feed, Stories, Reels, etc.) |

## Targeting & Assets

| Tool | Description |
|------|-------------|
| `browse_meta_targeting` | Browse available targeting options |
| `search_meta_targeting` | Search for specific targeting criteria |
| `discover_meta_assets` | Find reusable assets in your account |
| `validate_and_prepare_meta_assets` | Validate assets meet Meta's requirements |

## Lead Generation

| Tool | Description |
|------|-------------|

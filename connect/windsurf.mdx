# Connect Windsurf

Add the Adspirer MCP server to Windsurf's configuration.

<Steps>
  <Step title="Open MCP Settings">
    In Windsurf, go to **Settings → MCP Servers** and add a new server:

    - **Name:** Adspirer
    - **URL:** `https://mcp.adspirer.com/mcp`
    - **Transport:** HTTP
  </Step>
  <Step title="Authorize">
    Complete the OAuth 2.1 sign-in when prompted.
  </Step>
</Steps>

<Check>
**Verify:** Ask Windsurf "Show my campaign performance" to confirm the connection.
</Check>

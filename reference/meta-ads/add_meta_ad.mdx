<Badge color="yellow">Write</Badge>

User wants to add another ad/creative variation to an EXISTING ad set.

DO NOT USE to create a new campaign. For new campaigns:
- Video campaign - `create_meta_video_campaign`
- Image campaign - `create_meta_image_campaign`
- Carousel campaign - `create_meta_carousel_campaign`

Common scenarios:
- A/B test ad copy: Different headlines, primary text, or CTAs for the same audience
- A/B test creatives: Different images or videos for the same audience
- Format testing within one audience: Image vs video ad in the same ad set
- Dynamic Creative (DCO): Multiple images + text variations in one ad — Meta auto-optimizes

KEY DISTINCTION — add_meta_ad vs add_meta_ad_set:
- Same audience, different creative/copy - use THIS tool (add_meta_ad)
- Different audience/targeting - use `add_meta_ad_set` instead (creates a new ad set)

Ads in the same ad set SHARE: targeting, budget, schedule, pixel tracking.
Each ad has its OWN: creative (image/video/carousel), headline, primary text, CTA, landing page.

IMPORTANT: You need an ad_set_id from a previously created ad set. Use `list_meta_ad_sets` first if you don't have the ad_set_id.

## Parameters

<ParamField path="ad_set_id" type="string" required>
  The Meta Ad Set ID to add the ad to (required)
</ParamField>

<ParamField path="ad_type" type="string" required>
  Type of ad: 'image', 'video', or 'carousel'
</ParamField>

<ParamField path="image_url" type="string">
  Image URL (for image type)
</ParamField>

<ParamField path="existing_image_hash" type="string">
  Existing image hash
</ParamField>

<ParamField path="asset_bundle_id" type="string">
  Asset bundle ID
</ParamField>

<ParamField path="story_image_url" type="string">
  Image URL for Stories/Reels (9:16, recommended 1080x1920px). Uses asset_feed_spec for placement-specific creatives.
</ParamField>

<ParamField path="right_column_image_url" type="string">
  Image URL for Right Column (1.91:1, recommended 1200x628px). Uses asset_feed_spec for placement-specific creatives.
</ParamField>

<ParamField path="video_url" type="string">
  Video URL (for video type)
</ParamField>

<ParamField path="existing_video_id" type="string">
  Existing video ID
</ParamField>

<ParamField path="thumbnail_url" type="string">
  Custom thumbnail URL
</ParamField>

<ParamField path="cards" type="array of object">
  Carousel cards (2-10)
</ParamField>

<ParamField path="primary_text" type="string" required>
  Primary ad text (max 2200 chars, recommended 125). Supports emojis and line breaks.
</ParamField>

<ParamField path="headline" type="string">
  Headline (max 255 chars, recommended 40)
</ParamField>

<ParamField path="landing_page_url" type="string" required>
  Landing page URL (HTTPS)
</ParamField>

<ParamField path="call_to_action" type="string" default="&quot;LEARN_MORE&quot;">
  CTA button
</ParamField>

<ParamField path="description" type="string">
  Description (max 255 chars, recommended 30)
</ParamField>

<ParamField path="image_urls" type="array of string">
  List of image URLs for Dynamic Creative Optimization (2-10). Meta automatically tests all combinations and optimizes delivery. Mutually exclusive with image_url/existing_image_hash/story_image_url.
</ParamField>

<ParamField path="headlines" type="array of string">
  List of headline variations for Dynamic Creative (up to 5). Used with image_urls for DCO.
</ParamField>

<ParamField path="primary_texts" type="array of string">
  List of primary text variations for Dynamic Creative (up to 5). Used with image_urls for DCO.
</ParamField>

<ParamField path="descriptions" type="array of string">
  List of description variations for Dynamic Creative (up to 5). Used with image_urls for DCO.
</ParamField>

<ParamField path="lead_form_id" type="string">
  Lead form ID for OUTCOME_LEADS campaigns. If not provided, auto-fetched from the ad set's campaign.
</ParamField>

<ParamField path="facebook_page_id" type="string">
  Facebook Page ID
</ParamField>

<ParamField path="instagram_account_id" type="string">
  Instagram account ID
</ParamField>

<ParamField path="ad_name" type="string">
  Custom name for the ad. Also accepts 'name' as alias.
</ParamField>

<ParamField path="ad_account_id" type="string">
  Meta ad account ID
</ParamField>

<ParamField path="multi_advertiser" type="boolean">
  Set to false to opt out of Meta's multi-advertiser ad format at the creative level. Default (None) = Meta's default behavior (enrolled).
</ParamField>

## Example prompt

Ask your AI assistant (Claude, ChatGPT, Cursor, etc.) something like:

```
Add Meta Ad for my account.
```

Your AI client will call the `add_meta_ad` tool behind the scenes.

User wants to add a new ad set to an EXISTING campaign.

Common scenarios:
- Audience testing: Same ad format, different targeting/interests per ad set (e.g., "4 ad sets with different interests")
- Multi-format: Different ad types (image + video + carousel) under one campaign
- Budget split: Same creative, different budgets per audience segment
- Scaling: Adding new audiences/markets to an existing campaign
- Geographic split: Same ad, different locations per ad set
- Dynamic Creative (DCO): Set is_dynamic_creative=true, then use add_meta_ad with image_urls/headlines/primary_texts

CRITICAL: Each `create_meta_*_campaign` creates a NEW campaign.
To add more ad sets to the SAME campaign, you MUST use this tool.
NEVER call create_meta_*_campaign again for the same campaign.

IMPORTANT: You need a campaign_id from a previously created campaign. Use `get_meta_campaign_details` first if you don't have the campaign_id.

Supports all 3 ad types:
- `image`: Provide image_url, existing_image_hash, or asset_bundle_id
- `video`: Provide video_url or existing_video_id
- `carousel`: Provide cards array (2-10 cards)

## Parameters

<ParamField path="campaign_id" type="string" required>
  The Meta Campaign ID to add the ad set to (required). Get this from a prior create_meta_*_campaign call or get_meta_campaign_details.
</ParamField>

<ParamField path="ad_type" type="string" required>
  Type of ad: 'image', 'video', or 'carousel'
</ParamField>

<ParamField path="budget_daily" type="number">
  Daily budget in USD for this ad set (minimum $1). Required for non-CBO campaigns. Do NOT set for CBO campaigns — budget is managed at campaign level. Use daily_min_spend_target/daily_spend_cap instead.
</ParamField>

<ParamField path="campaign_budget_optimization" type="boolean" default="false">
  Set to true if the parent campaign uses Advantage Campaign Budget (CBO). When true, budget_daily is not required and bid_strategy is skipped on the ad set.
</ParamField>

<ParamField path="daily_min_spend_target" type="number">
  CBO only: minimum daily spend for this ad set in account currency.
</ParamField>

<ParamField path="daily_spend_cap" type="number">
  CBO only: maximum daily spend cap for this ad set in account currency.
</ParamField>

<ParamField path="image_url" type="string">
  Public URL of image to use (for image ad_type)
</ParamField>

<ParamField path="existing_image_hash" type="string">
  Existing Meta image hash (for image ad_type)
</ParamField>

<ParamField path="asset_bundle_id" type="string">
  Asset bundle ID from validate_and_prepare_meta_assets (for image ad_type)
</ParamField>

<ParamField path="video_url" type="string">
  Public URL of video to upload (for video ad_type)
</ParamField>

<ParamField path="existing_video_id" type="string">
  Existing Meta video ID (for video ad_type)
</ParamField>

<ParamField path="thumbnail_url" type="string">
  Custom thumbnail URL for video
</ParamField>

<ParamField path="cards" type="array of object">
  Carousel cards (2-10). Each: \{image_url, headline, landing_page_url, description?, call_to_action?\}
</ParamField>

<ParamField path="primary_text" type="string" required>
  Primary ad text (max 2200 chars, recommended 125). Supports emojis and line breaks.
</ParamField>

<ParamField path="headline" type="string">
  Headline (max 255 chars, recommended 40)
</ParamField>

<ParamField path="landing_page_url" type="string" required>
  Landing page URL (HTTPS)
</ParamField>

<ParamField path="call_to_action" type="string" default="&quot;LEARN_MORE&quot;">
  CTA button: LEARN_MORE, SHOP_NOW, SIGN_UP, etc.
</ParamField>

<ParamField path="description" type="string">
  Description (max 255 chars, recommended 30)
</ParamField>

<ParamField path="locations" type="array of any">
  Location targeting. Country codes (['US']) or location objects from search_meta_targeting ([\{'key': '2421836', 'type': 'city', 'radius': 25, 'distance_unit': 'mile'\}]). Default: ['US']
</ParamField>

<ParamField path="age_min" type="integer" default="18">
  Age Min
</ParamField>

<ParamField path="age_max" type="integer" default="65">
  Age Max
</ParamField>

<ParamField path="genders" type="array of string">
  Genders
</ParamField>

<ParamField path="interests" type="array of object">
  Interests
</ParamField>

<ParamField path="behaviors" type="array of object">
  Behaviors
</ParamField>

<ParamField path="custom_audiences" type="array of string">
  Custom Audiences
</ParamField>

<ParamField path="excluded_custom_audiences" type="array of string">
  Excluded Custom Audiences
</ParamField>

<ParamField path="lead_form_id" type="string">
  Lead form ID for OUTCOME_LEADS campaigns. If not provided, auto-fetched from existing campaign ads.
</ParamField>

<ParamField path="pixel_id" type="string">
  Meta Pixel ID for conversion tracking
</ParamField>

<ParamField path="pixel_event_name" type="string">
  Conversion event: PURCHASE, LEAD, ADD_TO_CART, OTHER, etc. Use OTHER with custom_conversion_id for custom events.
</ParamField>

<ParamField path="custom_conversion_id" type="string">
  Custom conversion ID for optimizing towards a specific custom conversion. When provided, pixel_event_name is ignored — Meta infers the event type from the custom conversion.
</ParamField>

<ParamField path="instagram_account_id" type="string">
  Instagram account ID
</ParamField>

<ParamField path="facebook_page_id" type="string">
  Facebook Page ID
</ParamField>

<ParamField path="ad_set_name" type="string">
  Custom name for the ad set
</ParamField>

<ParamField path="ad_name" type="string">
  Custom name for the ad created inside the ad set
</ParamField>

<ParamField path="ad_account_id" type="string">
  Meta ad account ID
</ParamField>

<ParamField path="multi_share_optimized" type="boolean" default="true">
  For carousel: optimize card order
</ParamField>

<ParamField path="budget_lifetime" type="number">
  Lifetime budget in account currency for this ad set (minimum $1). Mutually exclusive with budget_daily. Requires end_time to be set. Cannot be used with CBO campaigns.
</ParamField>

<ParamField path="end_time" type="string">
  End date/time for this ad set in ISO format (e.g., 2026-04-30T23:59:59). Required when using budget_lifetime. Optional with budget_daily.
</ParamField>

<ParamField path="multi_advertiser" type="boolean">
  Set to false to opt out of Meta's multi-advertiser ad format at the creative level. When false, sets contextual_multi_ads.enroll_status=OPT_OUT on the ad creative. Default (None) = Meta's default behavior (enrolled).
</ParamField>

<ParamField path="publisher_platforms" type="array of string">
  Which platforms to run on: ['facebook', 'instagram', 'audience_network', 'messenger']. Only listed platforms are included. Default: automatic (all).
</ParamField>

<ParamField path="facebook_positions" type="array of string">
  Facebook placements: feed, right_hand_column, story, facebook_reels, marketplace, search, etc. Only listed positions are included.
</ParamField>

<ParamField path="instagram_positions" type="array of string">
  Instagram placements: stream (feed), story, reels, explore, explore_home, ig_search, profile_feed. Only listed positions are included.
</ParamField>

<ParamField path="is_dynamic_creative" type="boolean">
  Set to true to enable Dynamic Creative Optimization on this ad set. Meta will test combinations of images, headlines, and text to find the best performer. Only 1 ad is allowed per DCO ad set. Use with add_meta_ad's image_urls/headlines/primary_texts fields.
</ParamField>

<ParamField path="destination_type" type="string">
  Override the ad set destination type. Set to 'WEBSITE' for OUTCOME_LEADS campaigns that track conversions on your website via Meta Pixel instead of using Meta's built-in lead form. Options: WEBSITE, ON_AD, WEBSITE_AND_ON_AD.
</ParamField>

## Example prompt

Ask your AI assistant (Claude, ChatGPT, Cursor, etc.) something like:

```
Add Meta Ad Set for my account.
```

Your AI client will call the `add_meta_ad_set` tool behind the scenes.

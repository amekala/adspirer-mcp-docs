The Adspirer MCP server exposes **45** Google Ads tools to AI assistants via the Model Context Protocol. This page lists every tool with links to its dedicated reference.

## Read-only tools

| Tool | Description |
|------|-------------|
| [`analyze_search_terms`](/reference/google-ads/analyze_search_terms) | Discover keyword opportunities and optimize match types by analyzing actual search terms. |
| [`analyze_wasted_spend`](/reference/google-ads/analyze_wasted_spend) | Analyze wasted ad spend and identify campaigns losing money or underperforming. |
| [`discover_existing_assets`](/reference/google-ads/discover_existing_assets) | 🔍 Discover existing assets in the Google Ads account (images, sitelinks, callouts, structured snippets). |
| [`explain_performance_anomaly`](/reference/google-ads/explain_performance_anomaly) | Explain why a performance metric changed using statistical analysis and historical context. |
| [`get_campaign_performance`](/reference/google-ads/get_campaign_performance) | Analyze Google Ads campaign performance with comprehensive insights and recommendations. |
| [`get_campaign_structure`](/reference/google-ads/get_campaign_structure) | Get campaign structure with ad groups, keywords, ads, and extensions. |
| [`get_campaign_targeting`](/reference/google-ads/get_campaign_targeting) | Get location/geo targeting for a Google Ads campaign. |
| [`get_pmax_audience_signals`](/reference/google-ads/get_pmax_audience_signals) | Get current audience signals for a Performance Max campaign. |
| [`get_pmax_search_themes`](/reference/google-ads/get_pmax_search_themes) | Get current search themes for a Performance Max campaign. |
| [`list_campaign_extensions`](/reference/google-ads/list_campaign_extensions) | List all extensions (sitelinks, callouts, structured snippets) for a campaign. |
| [`list_campaigns`](/reference/google-ads/list_campaigns) | List all Google Ads campaigns for the connected account. |
| [`optimize_budget_allocation`](/reference/google-ads/optimize_budget_allocation) | Optimize budget allocation across campaigns using linear programming to maximize conversions. |
| [`research_keywords`](/reference/google-ads/research_keywords) | Research high-intent keywords using Google Keyword Planner API. |
| [`search_audiences`](/reference/google-ads/search_audiences) | Search for available audience segments (in-market, affinity, custom). |
| [`select_google_campaign_type`](/reference/google-ads/select_google_campaign_type) | USE THIS TOOL FIRST WHEN: User wants to create a Google Ads campaign but hasn't specified the campaign type… |
| [`suggest_ad_content`](/reference/google-ads/suggest_ad_content) | Generate AI-suggested headlines and descriptions based on campaign keywords. |
| [`validate_video`](/reference/google-ads/validate_video) | Validate video for ad campaigns (unified tool for all platforms). |

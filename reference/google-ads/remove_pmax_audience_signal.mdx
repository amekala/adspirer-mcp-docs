<Badge color="yellow">Write</Badge> <Badge color="red">Destructive</Badge>

Remove a specific audience signal from a Performance Max campaign.

Use get_pmax_audience_signals first to get the signal_resource_name.

**Parameters:**
- campaign_id: The PMax campaign ID
- signal_resource_name: Resource name from get_pmax_audience_signals

**Execution time:** 2-5 seconds

## Parameters

<ParamField path="campaign_id" type="string" required>
  The Google Ads campaign ID (numeric). Example: '21854471508'
</ParamField>

<ParamField path="signal_resource_name" type="string" required>
  The resource name of the audience signal to remove. Get from get_pmax_audience_signals tool. Example: 'customers/1234567890/assetGroupSignals/123~456'
</ParamField>

<ParamField path="customer_id" type="string">
  Google Ads customer ID. Required for multi-account users. Get from list_connected_accounts.
</ParamField>

## Example prompt

Ask your AI assistant (Claude, ChatGPT, Cursor, etc.) something like:

```
Remove PMax Audience Signal for my account.
```

Your AI client will call the `remove_pmax_audience_signal` tool behind the scenes.

Get current search themes for a Performance Max campaign.

Returns each search theme's text and approval_status (APPROVED, LIMITED, DISAPPROVED, UNDER_REVIEW).

**Use when:**
- User wants to see what search themes are set on their PMax campaign
- Before adding new themes (to check current count — max 50)
- To verify theme approval status after adding

**Parameters:**
- campaign_id: The PMax campaign ID (get from list_campaigns)

**Execution time:** 1-2 seconds

## Parameters

<ParamField path="campaign_id" type="string" required>
  The Google Ads campaign ID (numeric). Example: '21854471508'
</ParamField>

<ParamField path="customer_id" type="string">
  Google Ads customer ID. Required for multi-account users. Get from list_connected_accounts.
</ParamField>

## Example prompt

Ask your AI assistant (Claude, ChatGPT, Cursor, etc.) something like:

```
Get PMax Search Themes for my account.
```

Your AI client will call the `get_pmax_search_themes` tool behind the scenes.

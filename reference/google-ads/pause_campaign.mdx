<Badge color="yellow">Write</Badge>

Quickly pause a running campaign.

Pausing a campaign stops all ads from showing immediately.
This is REVERSIBLE using resume_campaign.

**Parameters:**
- campaign_id: The campaign ID (REQUIRED - get from list_campaigns)
- customer_id: Optional (uses connected account if omitted)

**Execution time:** 1-2 seconds

**Returns:**
- Campaign name
- Before status (e.g., ENABLED)
- After status (PAUSED)
- Confirmation message

**When to use:**
- User says "pause the campaign", "stop the ads", "turn it off"
- User wants to temporarily stop spending
- User needs to make changes before ads continue

## Parameters

<ParamField path="campaign_id" type="string" required>
  The campaign ID to pause. Use list_campaigns first to get available campaign IDs.
</ParamField>

<ParamField path="customer_id" type="string">
  Google Ads customer ID. Required for multi-account users. Get from list_connected_accounts.
</ParamField>

## Example prompt

Ask your AI assistant (Claude, ChatGPT, Cursor, etc.) something like:

```
Pause Campaign for my account.
```

Your AI client will call the `pause_campaign` tool behind the scenes.

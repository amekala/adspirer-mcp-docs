## Read-only tools

| Tool | Description |
|------|-------------|
| [`get_business_profile`](/reference/business-profile/get_business_profile) | Get the user's business profile for contextual recommendations. |

## Write tools

| Tool | Description |
|------|-------------|
| [`infer_business_profile`](/reference/business-profile/infer_business_profile) | Automatically infer business profile from campaign data using AI analysis. |
| [`save_business_profile`](/reference/business-profile/save_business_profile) | Save or update the user's business profile with provided details. |

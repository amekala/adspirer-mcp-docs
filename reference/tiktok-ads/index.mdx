The Adspirer MCP server exposes **1** TikTok Ads (Router) tool to AI assistants via the Model Context Protocol. This page lists every tool with links to its dedicated reference.

<Note>
This is a **federated router tool**. Individual TikTok Ads actions are invoked through this meta-tool. Call it with `action="list_tools"` to discover every available action, then `action="execute"` with a `tool_name` and `arguments` to run one.
</Note>
